    VuePress + Github pages 搭建个人博客

    # VuePress + Github pages 搭建个人博客

    参考文章：https://juejin.cn/post/7041134607869149215

    # 安装Homebrew

    参考链接：https://www.jianshu.com/p/e0471aa6672d?utm_campaign=hugo

    已安装的可以略过

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

    # 安装npm

    brew install node

    安装好后查看下node和npm的版本

    node -v
npm -v

    # 安装yarn

    个人喜好，也可以不安装，选择用npm（这里只演示yarn命令的方式）

    npm install yarn

    同样，查看下yarn的版本

    yarn -v

    接下来正式搭建博客

    首先创建一个存放文件的新目录并进入该目录

    mkdir MyBlog && cd MyBlog

    对项目进行初始化

    yarn init

    安装vuepress

    yarn add -D vuepress

    创建docs文件夹

    mkdir docs

    在文件夹下新增一个文件

    vim README.md

    在package.json中添加启动和构建命令

    {
    "scripts": {
        "docs:dev": "vuepress dev docs",
        "docs:build": "vuepress build docs"
    }
}

    在文档目录下创建一个.vuepress的目录，所有vuepress相关的文件都会被放在这里。

    在.vuepress文件夹下添加config.js，配置博客的标题和描述。

    module.exports = {
    title: "raferlyan's blog",
    description: 'some technical articles on programming, especially backend',
}

    此时目录结构如下：

    ├─ docs
│  ├─ README.md
│  └─ .vuepress
│        └─ config.js

    本地启动

    yarn docs:dev

    vuepress会在http://localhost:8080启动一个热加载的开发服务器。

    yarn docs:dev
sh deploy.sh
